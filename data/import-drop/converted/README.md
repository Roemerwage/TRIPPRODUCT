# Converted Files

These files were extracted from your dropped TypeScript source files and are ready for admin import.

- `trip-import.tsv` -> upload as **Trip Spreadsheet (required)**
- `places-import.tsv` -> upload as **Places Spreadsheet (optional)**

Import path in admin:

1. Open `Trip Console`
2. Go to `Spreadsheet Import`
3. Upload `trip-import.tsv` (and `places-import.tsv` if you want places)
4. Choose merge or replace mode
5. Import into editor

Notes:
- Keep `places-import.tsv` for optional POI/tips/logistics only.
- Do not duplicate accommodations or main itinerary activities in places.
